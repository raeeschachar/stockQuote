### Pre Requisites
- Python3

### Steps
- Open terminal 
```
$ mkdir (name of folder)
$ cd (name of folder)
$ git clone git@github.com:raeeschachar/stockQuote.git
$ cd stockQuote
$ pip3 install -r requirements.txt
```
